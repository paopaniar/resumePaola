# CV — Paola Paniagua Arroyo

Currículum web estático (HTML, CSS y JavaScript).

## Ejecutar en local

Desde esta carpeta:

```powershell
cd "c:\Users\User\Downloads\cv"
python -m http.server 8765
```

Abre en el navegador: **http://localhost:8765**

Para detener el servidor: `Ctrl+C` en la terminal.

## Archivos

| Archivo      | Descripción                          |
| ------------ | ------------------------------------ |
| `index.html` | Contenido del CV (ES / EN)           |
| `style.css`  | Estilos                              |
| `script.js`  | Idioma, descarga PDF y TXT           |
